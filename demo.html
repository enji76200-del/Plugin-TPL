Fonctionnalités du Plugin

  • Navigation par date : Utilisez les flèches ou les touches fléchées du clavier pour naviguer entre les dates
  • Période : Affiche 9 jours consécutifs (aujourd'hui + 8 jours)
  • Colonnes : Date, Horaires, Créneaux, et Actions
  • Ajout de créneaux : Cliquez sur "+" ou "Ajouter un créneau" pour créer de nouveaux créneaux
  • Suppression : Cliquez sur "×" pour supprimer un créneau
  • Inscription : Cliquez sur une case "Disponible" pour vous inscrire
  • Style Excel : Design similaire à Microsoft Excel avec bordures et couleurs

Utilisation dans WordPress

Utilisez le shortcode [registration_table] pour afficher le tableau sur n'importe quelle page ou article.
